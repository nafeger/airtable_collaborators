This project will query the airtable API and build out an airtable base with the users accessing your system.



Working on the Repo:

one time:
```bash
python3.7 -m venv venv/
source venv/bin/activate
cd app
pip install -r requirements.txt
```

each time:
```bash
nvm use v8.11.3
source venv/bin/activate
```
